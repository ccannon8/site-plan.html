Overview

Purpose

The purpose of this site is to help users track their mood and productivity levels, gain insights from trends, and support mindfulness and healthy habits. It offers daily logs, visual summaries, and motivational feedback.

Audience

The site is designed for students, remote workers, and individuals interested in improving mental health, productivity, and personal well-being.

Dynamic elements

JavaScript will allow users to:

  • Log mood and productivity using sliders or dropdowns
  • See daily entries rendered dynamically from a stored array of objects
  • Use localStorage to persist user data between sessions
  • Filter past logs and visualize trends using graphs
  • Receive motivational messages based on mood scores
Code will use functions, DOM manipulation, conditionals, ES Modules, and array methods like filter, map, and forEach.
